# x-running

## Acesse o App

Você pode acessar o app diretamente no seu dispositivo móvel (iPhone, Android) ou computador pelo navegador:

👉 [Abrir o x-running](https://whimsical-capybara-8c382a.netlify.app)

### Dica: Adicione o app à tela inicial do seu iPhone

Para uma experiência semelhante a um aplicativo nativo no iPhone:

1. Abra o link acima no Safari.
2. Toque no ícone de compartilhamento (quadrado com seta para cima).
3. Selecione “Adicionar à Tela de Início”.
4. Escolha um nome e toque em “Adicionar”.

Pronto! Agora você pode acessar o x-running como se fosse um app instalado no seu iPhone.
